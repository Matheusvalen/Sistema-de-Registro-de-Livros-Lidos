Sobre o Sistema

Este sistema foi criado para ajudar na organização de livros lidos. Ideal para leitores, bibliotecas e escolas, ele permite o registro e consulta rápida de livros de forma simples e eficiente.


Para acessar o sistema clique em Cadastro no botão acima. Caso já tenha se cadastrado clique em login no botão acima
